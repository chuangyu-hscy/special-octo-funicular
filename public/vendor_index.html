To obtain the order or change the status/location of each van, please add the unique VAN ID after the URL

Example: https://snacks-in-a-van-4399.herokuapp.com/vendor/Genevieve%20Adele to obtain van info

Using /van_id/orders to obtain the list of all orders

Using /van_id/completed to obtain the list of completed orders

Using /van_id/outstanding to obtain the list of outstanding orders

Using /van_id/fulfilled to obtain the list of fulfilled orders

Using /van_id/canceled to obtain the list of canceled orders

Using /van_id/turnOn to mark van ready-for-order

Using /van_id/turnOff to mark van off

Posting a JSON at /van_id/setLocation to set the location for van

Posting a JSON at https://snacks-in-a-van-4399.herokuapp.com/vendor/setStatus to set the status for a van
